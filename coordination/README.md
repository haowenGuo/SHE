# Coordination

This directory is the shared operational memory for multi-Codex development.

Use it like this:

1. Check [TASK_BOARD.md](./TASK_BOARD.md)
2. Update [STATUS_LEDGER.md](./STATUS_LEDGER.md)
3. Work inside one bounded workstream
4. Submit handoff details using the templates in this directory
5. Record integration impact in [INTEGRATION_REPORT.md](./INTEGRATION_REPORT.md)

Subdirectories:

- `WORKSTREAMS/` for per-workstream specs
- `HANDOFFS/` for completed handoff notes
